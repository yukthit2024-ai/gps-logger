You are an expert Android developer. Build a complete Android application in **Java (not Kotlin)** using Android Studio.

## App Overview

Create a simple app with one screen containing two buttons:

* "Start Tracking"
* "End Tracking"

## Functional Requirements

### Start Tracking:

* When user clicks "Start Tracking":

  * Check for location permission (ACCESS_FINE_LOCATION)
  * If not granted, request it at runtime
  * Once granted:

    * Start GPS/location tracking using FusedLocationProviderClient
    * Capture:

      * Latitude
      * Longitude
      * Timestamp (ISO 8601 format)
    * Collect updates every 5–10 seconds
    * Store the collected data in memory as a JSON array

### End Tracking:

* When user clicks "End Tracking":

  * Stop location updates immediately
  * Save all collected data into a JSON file
  * File must be properly closed and flushed

## Timestamp Format

* Store timestamp in ISO 8601 format, e.g.:
  "2026-04-12T14:30:00Z"
* Use Java time APIs (java.time package), such as:

  * Instant.now()
  * DateTimeFormatter.ISO_INSTANT

## JSON Format

Store data like:
[
{
"latitude": 12.9716,
"longitude": 77.5946,
"timestamp": "2026-04-12T14:30:00Z"
}
]

## File Storage

* Save file in app-specific external storage:
  getExternalFilesDir(null)
* File name format:
  location_logs_<current_timestamp>.json
* Ensure compatibility with scoped storage (no MANAGE_EXTERNAL_STORAGE)
* Log the saved file path using Logcat

* Use ConstraintLayout

* Initially:
